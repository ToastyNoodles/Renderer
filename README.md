# 2024 Renderer
## Features
* Materials
  * With diffuse, normal, roughness, metallic, and ambient occlusion mapping.
* Light Types
* Point Light
* Directional Light
* Model Loading
* Deferred Rendering
* Transparency
* Cubemaps
* Diffuse Image Based Lighting
