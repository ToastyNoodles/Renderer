# 2024 Renderer
## Features
* Materials
  * With diffuse, normal, roughness, metallic, and ambient occlusion mapping.
* Point lights and directional lights
* Model loading
* Deferred rendering
* Transparency
* Cubemaps
* Diffuse Image Based Lighting
